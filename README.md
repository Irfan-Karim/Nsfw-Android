# NSFW - Not Safe For Work Detector Android

## Import

### Add it in your root build.gradle at the end of repositories:

```
allprojects {
  repositories {
  ...
  maven { url 'https://jitpack.io' }
  }
}
```

### Add the dependency

```
dependencies {
  implementation 'com.github.Irfan-Karim:Nsfw-Android:1.0.0'
}
```
